人間ゲート

  • 出音の最終音量、マスタリング感、低域の圧。音割れ、limiter pumping、平坦化がないこと。
  • techno / acid の「気持ちよさ」と「やりすぎ」の境界。
  • piano が Music に残るべきか、chill に寄せるべきか。
  • OpenClaw という repo 名を残すか、将来 rename / alias するか。
  • Desk が次PR候補を出すだけでよいか、将来 orchestrator runtime に進めるか。

触らないもの

  • raw audio、sample、lyrics、外部音源。
  • dependency、schema、release設定。
  • 自動PR、自動merge、自動archive。
  • drum-floor、namima、chill、openclaw 側の実装。
