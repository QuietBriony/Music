## 01 TABASCO — 0:44 · 136 BPM · D minor · opener

**Theme:** band manifesto. Short, sharp, chant-driven. Like a callsign.

```
(spoken intro — single voice, dry)
This is the call.
Wake up, listen up, line it up.

(full band, chant × 4)
TAB-AS-CO!  one two three four
TAB-AS-CO!  here we go
TAB-AS-CO!  open the door
TAB-AS-CO!  here we go
```

Performance: spoken intro low + dry, chant high + reverb-soaked.
Each "TABASCO" pronounced as 3 hard syllables ta-bas-co.
