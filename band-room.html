URL を 1 つ以上入れて追加すると、kit selector の online セクション末尾に並ぶ。localStorage に保存されて次回も使える。snare URL のみ入れて kick/hat は空でも OK (空 voice は無音)。

per-voice override: 各 voice (kick/snare/hat/...) を別 kit から個別に取れる。kick = 808 / snare = acoustic / hat = 909 みたいなミックス組める。

🎤 mic follow groove (演奏に drum 追従)
disabled

マイクで gtr / bass / vocal を聞いて、演奏が激しいときは drum の velocity を上げ (vel +30%)、静かなときは下げる (vel -30%)。録音 はしない、解析だけ。AI 再現モードで drum 動的に呼吸する。影響度 0% で 無効化、100% で最大変動 ±30%。

🐢 practice tempo + swing

tempo: 原曲 BPM × mult。50% = ハーフテンポ、120% = 早回し。原音モードでは pitch も変わるので AI 再現モード推奨。
swing: 8 分音符の偶数を遅らせる shuffle 量。25% でジャズ三連符寄り、50% で完全 swing。lofi-nujabes profile なら 20-30% が気持ちいい。
